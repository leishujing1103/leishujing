- 最简单的排序算法 —— 冒泡 bubbleSort
  排序的过程，两两相比，如果 j 小于 i 位置的数，互换 i，j
  两重循环
  一重循环 一定可以得到最小值 在相应的位置上
  每次解决一个位置
  i = 0       i < length - 1    i++
  j = i + 1   j < length        j++
  时间复杂度 n^2(平方)
